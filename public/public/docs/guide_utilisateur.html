Guide d'utilisation

Application : Carte interactive – Version 1.0

1. Introduction

Cette application permet de consulter une carte interactive, d'afficher des marqueurs et de visualiser des informations associées (catégorie, description, etc.).

Ce guide s'adresse aux utilisateurs réguliers de l'application et explique les principales fonctionnalités disponibles.

2. Accès à l'application

  • Aucune connexion n'est requise pour un utilisateur qui n'est pas un administrateur du site

3. Navigation dans l'interface

3.1 Carte principale

  • La carte occupe la partie centrale de l'écran.
  • Utilisez la souris ou les boutons +/- pour zoomer et dézoomer.
  • Cliquez-glissez pour déplacer la carte.

3.2 Panneau de filtres / menu

  • Un panneau latéral ou un menu permet de filtrer les marqueurs (par catégorie, par état, etc.).
  • Vous pouvez activer/désactiver certaines catégories pour réduire l'affichage.

4. Consultation des marqueurs

  • Cliquez sur un marqueur pour ouvrir sa fiche d'information.
  • La fiche affiche généralement : le titre, la catégorie, la description et, au besoin, des images.
  • Fermez la fiche pour revenir à la vue complète de la carte.

5. Modification des marqueurs

  • Il est possible pour l'utilisateur de cliquer sur l'un des marqueur disponible sur la carte et de proposer une modification s'il le souhiate, la demande sera ensuite envoyée à un administrateur pour validation.
  • Il est possible pour l'utilisateur d'ajouter un commentaire (témoignage) lors du clic sur un marqueur disponible sur la carte, le commentaire sera ensuite envoyé à un administrateur pour validation.
  • Il est possible pour l'utilisateur d'ajouter des image

6. Problèmes fréquents

  • La carte ne se charge pas : vérifiez votre connexion Internet, puis rechargez la page.
  • Aucun marqueur n'apparaît : vérifiez que les filtres ne sont pas trop restrictifs.
  • Message d'erreur de connexion : vérifiez vos identifiants ou contactez un administrateur.
