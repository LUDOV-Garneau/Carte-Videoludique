Guide d'utilisation – Administrateur

Application : Le jeu vidéo au Québec – Interface d'administration

1. Rôle de l'administrateur

L'administrateur est responsable de la modération et de la qualité des données affichées sur la carte « Le jeu vidéo au Québec ». Il ou elle :

  • valide les demandes d'ajout et de modification de lieux ;
  • approuve ou refuse les commentaires ;
  • gère les marqueurs archivés ;
  • ajoute de nouveaux marqueurs ;
  • gère les catégories utilisées sur la carte.

Seuls les utilisateurs possédant un compte avec droits d'administration ont accès à cette interface (bouton Admin et vue d'administration).

2. Accès à l'interface d'administration

  • Connectez-vous à l'application avec votre compte administrateur.
  • La page admin est disponible en ecrivant "admin" dans l'url du site, cela vous dirigera vers connexion si non connecté
  • Vous arrivez sur la page Journal d'activité.

Dans le bandeau supérieur, vous pouvez également accéder à :

  • Admin : affiche les information du compte admin connecté
  • Administrateurs : gestion ou liste des comptes administrateurs (modification et suppression) ;
  • Créer un compte : ajout d’un nouveau compte administrateur ;
  • Se déconnecter : fermeture de la session actuelle.

3. Journal d'activité

Le bloc supérieur de la page présente le Journal d'activité sous forme d’onglets. Chaque onglet correspond à un type d’élément à modérer.

3.1 Les onglets de modération

  • Demande d'ajout : lieux proposés par les utilisateurs pour être ajoutés à la carte.
  • Demande de modification : modifications suggérées sur des lieux déjà existants (ex. correction d'adresse, description, catégorie, etc.).
  • Commentaires à approuver : commentaires laissés par les utilisateurs, en attente de validation avant d’être visibles sur la carte.
  • Marqueurs archivés : liste des lieux qui ont été archivés (non visibles sur la carte publique, mais conservés pour référence).
  • Commentaires archivés : commentaires retirés de l’affichage public.

3.2 Tableau de modération

Pour les demandes d'ajout et de modification, un tableau affiche la liste des éléments. Les colonnes principales sont :

  • Lieu : nom du lieu (ex. salle, boutique, événement, etc.).
  • Adresse : adresse complète du lieu.
  • Info : lien Afficher la description pour consulter la description détaillée proposée par l'utilisateur.
  • Modification : bouton Modifier pour ajuster le contenu avant de l'accepter (ex. corriger l’orthographe, ajuster la catégorie, etc.).
  • Accepter : bouton turquoise permettant de valider la demande.
  • Refuser : bouton rouge permettant de rejeter la demande.

3.3 Accepter ou refuser une demande

Pour chaque ligne du tableau :

  1. Cliquez sur Afficher la description pour lire les détails du lieu.
  2. Si nécessaire, cliquez sur Modifier pour ajuster les informations (nom, adresse, catégorie, description, etc.).
  3. Cliquez sur Accepter pour valider l’ajout ou la modification.
    Le marqueur sera alors mis à jour sur la carte.
  4. Cliquez sur Refuser si la demande est incomplète, non pertinente ou ne respecte pas les critères de la carte.

3.4 Tri des éléments

En haut à droite du tableau, un menu Trier par permet d’ordonner les éléments, par exemple « Plus récent d'abord ». Utilisez ce tri pour traiter en priorité les demandes les plus récentes.

4. Actions sur la carte

Sous le tableau du journal d'activité, la carte affiche les marqueurs géolocalisés. Elle permet de visualiser rapidement la position des lieux en cours de traitement ou déjà présents dans la base.

4.1 Bouton « + Ajouter un marqueur »

Le bouton turquoise + Ajouter un marqueur, situé au-dessus de la carte (côté droit), permet de créer un lieu directement en tant qu'administrateur.

  1. Cliquez sur + Ajouter un marqueur.
  2. Cliquez sur la carte pour positionner le marqueur à l’emplacement souhaité.
  3. Remplissez les informations demandées (titre, adresse, description, catégorie, etc.).
  4. Enregistrez pour ajouter le lieu à la carte.

4.2 Bouton « Gérer les catégories »

Le bouton Gérer les catégories ouvre l’interface de gestion des catégories utilisées pour classer les lieux.

  • Ajouter une nouvelle catégorie (ex. Arcade, Magasin spécialisé, etc.).
  • Renommer ou corriger une catégorie existante.
  • Supprimer ou désactiver une catégorie qui n’est plus utilisée.

Assurez-vous de garder une liste de catégories claire et cohérente pour faciliter la recherche et la compréhension des utilisateurs.

5. Bonnes pratiques de modération

  • Vérifier rapidement les demandes d’ajout pour éviter les doublons.
  • Lire attentivement la description et l’adresse pour s’assurer que le lieu correspond bien au thème « jeu vidéo au Québec ».
  • Utiliser le bouton Modifier pour harmoniser les titres (orthographe, casse, uniformité des noms).
  • Archiver plutôt que supprimer définitivement lorsqu’un lieu n’est plus d’actualité, mais qu’il peut avoir une valeur historique.

6. Support et évolution de l'application

Pour toute anomalie technique (erreurs d’affichage, lenteurs, bugs) ou pour demander une nouvelle fonctionnalité, contactez l’équipe de développement responsable de la carte.

Notez si possible :

  • la date et l’heure du problème ;
  • la page ou l’onglet concernés (ex. « Demande d’ajout », « Marqueurs archivés ») ;
  • les actions effectuées juste avant l’apparition du problème.
